The v1 JSON schema and issue code catalog are public contract candidates until `v1.0.0`.

JSON Schema

The versioned schema lives at schemas/v1/devdoctor-output.schema.json. The top-level alias schemas/devdoctor-output.schema.json points to the latest schema.

Issue Codes

The machine-readable catalog lives at schemas/v1/issue-codes.json. The Markdown list is maintained at docs/issue-codes.md.

Compatibility Policy

  • Before `v1.0.0`, schema v1 and issue codes are candidates.
  • After `v1.0.0`, schema v1 will not receive breaking changes during `v1.x`.
  • Issue codes will not be removed or change meaning without deprecation.
  • New issue codes may be added.
