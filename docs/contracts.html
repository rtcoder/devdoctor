DevDoctor docs

Contracts

The v1 JSON schema and issue code catalog are stable public contracts for DevDoctor v1.x.

JSON Schema

The versioned schema lives at schemas/v1/devdoctor-output.schema.json. The top-level alias schemas/devdoctor-output.schema.json points to the latest schema.

Issue Codes

Browse the human-readable catalog at Issue Codes. The machine-readable catalog remains available at schemas/v1/issue-codes.json for automation.

Compatibility Policy

  • Schema v1 will not receive breaking changes during v1.x.
  • Issue codes will not be removed or change meaning without deprecation.
  • New issue codes may be added.
