Release Verification

Tagged releases publish a PHAR, SHA-256 checksum, Cosign signature, and Sigstore certificate.

HomeInstallationSafety

Checksum

sha256sum --check devdoctor.phar.sha256

Signature

cosign verify-blob \
  --certificate devdoctor.phar.pem \
  --signature devdoctor.phar.sig \
  --certificate-identity-regexp 'https://github.com/rtcoder/devdoctor/' \
  --certificate-oidc-issuer https://token.actions.githubusercontent.com \
  devdoctor.phar

Release Workflow

The release workflow validates tag and Composer version alignment, runs tests and formatting, builds the PHAR on PHP 8.5, smoke tests it, signs it, and publishes release assets.
