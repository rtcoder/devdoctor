DevDoctor docs

Configuration

devdoctor.yml currently configures environment rules under modules.env.

Generate a Config

php devdoctor init
php devdoctor init --dry-run
php devdoctor init --config=config/devdoctor.yml

The wizard previews YAML and writes only after confirmation. In CI or --no-interaction mode it can only run with --dry-run.

Example

modules:
  env:
    files:
      env: .env
      example: .env.example
    rules:
      APP_KEY:
        required: true
      APP_DEBUG:
        type: bool
      APP_URL:
        type: url
      CACHE_DRIVER:
        allowed:
          - file
          - redis
      REDIS_URL:
        required_when:
          CACHE_DRIVER: redis
